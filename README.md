# Network Editor Web

Packet Tracer-style network editor built with React, TypeScript, and Vite.

## Local Development

```bash
npm install
npm run dev
```

The app supports an optional local `id.txt` file for a built-in login account. Copy `id.example.txt` to `id.txt` and edit the values if you want to use that shortcut locally.

```txt
id: your-id
pw: your-password
```

`id.txt` is intentionally ignored so credentials are not committed.

## Build

```bash
npm run build
```
